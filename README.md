# 💱 Currency Converter (API Based)

A simple and responsive **Currency Converter Web App** built using HTML, CSS, and JavaScript with a real-time exchange rate API.

---

## 🚀 Features

- 🌍 Real-time currency conversion using API
- 💱 Convert between multiple currencies
- ⚡ Fast and lightweight UI
- 📱 Fully responsive design
- 🔄 Auto-updated exchange rates
- 🎯 Simple and user-friendly interface

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Exchange Rate API (e.g. ExchangeRate API / Fixer / etc.)
